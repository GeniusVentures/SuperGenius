Detailed Description

template<typename Key, typename Type, typename... Arguments>
class sgns::subscription::Subscriber< Key, Type, Arguments >

Is a wrapper class, which provides subscription to events from SubscriptionEngine

Template Parameters
Keyis a type of a subscription Key.
Typeis a type of an object to receive notifications in.
Argumentsis a set of types of objects needed to construct Type.
