An index of the individual authority in the current authority list that should be immediately disabled until the next authority set change. When an authority gets disabled, the node should stop performing any authority functionality from that authority, including authoring blocks and casting FINALITY votes for finalization. Similarly, other nodes should ignore all messages from the indicated authority which pretain to their authority role.orce an authority set change after the given delay of N blocks, is an imported block which has been validated by the block production conensus engine.
